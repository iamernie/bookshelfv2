# Screenshots Needed for Documentation

This file lists all the screenshots referenced in the BookShelf V2 documentation. Please capture these screenshots and save them to the `docs/images/` directory with the exact filenames listed below.

---

## Screenshot Guidelines

- **Format:** PNG preferred (for crisp UI elements)
- **Resolution:** 1200-1600px wide for full-page shots, 600-800px for modals
- **Theme:** Capture in both light and dark mode if possible (light as primary)
- **Content:** Use realistic sample data (book titles, authors, etc.)
- **Privacy:** Avoid any personal information in screenshots

```bash

  -pointsize 24 -annotate 0 "dashboard-overview.png" \
  docs/images/dashboard-overview.png
```

Or use online placeholder services:
- https://placeholder.com/
- https://placehold.co/
