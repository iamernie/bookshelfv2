Magic Shelves are saved searches that automatically populate with matching books.

---

## What Are Magic Shelves?

Define filter criteria → Shelf updates automatically when books match.

No manual book-adding required!

| Feature | Magic Shelves | Tags |
|---------|---------------|------|
| Population | Automatic | Manual |
| Updates | Dynamic | Static |
| Criteria | Multiple filters | Single label |

---

## Creating a Magic Shelf

1. Navigate to **Shelves**
2. Click **+ New Shelf**

![Create Magic Shelf](../images/magic-shelf-create.png)
*Define your filter rules*

3. Name it, pick an icon
4. Set your filter rules
5. Click **Create**

---

## Filter Options

| Filter | Example |
|--------|---------|
| **Status** | Read, Currently Reading |
| **Genre** | Fantasy, Sci-Fi |
| **Author** | Brandon Sanderson |
| **Series** | Stormlight Archive |
| **Format** | Audiobook |
| **Rating** | 4-5 stars |
| **Tags** | favorites |
| **Year Read** | 2024 |
| **Page Count** | 500+ pages |

Filters combine with **AND** logic.

---

## Shelf Ideas

| Name | Rules |
|------|-------|
| **5-Star Fantasy** | Genre: Fantasy + Rating: 5 |
| **Quick Reads** | Pages: 1-200 + Status: To Read |
| **Book Club** | Tags: book-club |
| **Epic Reads** | Pages: 600+ + Rating: 4-5 |
| **Comfort Reads** | Rating: 5 + Tags: cozy |
| **TBR Under 300** | Status: To Read + Pages: 1-300 |

---

## Public Shelves

Share your curated collections:

1. Edit shelf → Toggle **Public**
2. Copy the share URL
3. Anyone can view (but not edit)

---

## Managing Shelves

- **Edit** — Change name, icon, or rules
- **Duplicate** — Copy as starting point
- **Reorder** — Drag in sidebar
- **Delete** — Removes shelf only (books stay!)

Shelves appear in the sidebar for quick access.
