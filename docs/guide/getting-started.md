## Prerequisites

Choose your deployment method:

### Docker (Recommended)
- Docker and Docker Compose installed
- 1GB RAM minimum
- 5GB disk space (more for large libraries)

### Local Development
- Node.js 18+ and npm
- SQLite3
- Linux, macOS, or WSL2 on Windows
