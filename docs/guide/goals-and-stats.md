## Reading Goals

### Goal Types

| Type | Description | Example |
|------|-------------|---------|
| **Total Books** | Read X books this year | 52 books in 2024 |
| **Genre Challenge** | Focus on a genre | 12 Fantasy books |
| **Author Challenge** | Deep dive an author | 5 Sanderson books |
| **Format Goal** | Try a format | 10 audiobooks |
| **Page Count** | Total pages | 20,000 pages |
| **Monthly Target** | Consistent pace | 4 books/month |

### Creating a Goal

1. Navigate to **Stats â†’ Goals**
2. Click **+ New Goal**

![Create Goal](../images/create-goal.png)
*Set your reading challenge*

3. Choose type, set target, select time period
4. Click **Create**
