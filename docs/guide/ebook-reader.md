|--------|-----------|----------|

| **PDF** | `.pdf` | Fixed-layout, textbooks |
| **CBZ** | `.cbz` | Comics, manga |

## Uploading Ebooks

### From the Book Modal

1. Open any book
2. Go to **Ebook** tab
3. Drag & drop your file (or click to browse)
4. Metadata is extracted automatically!

### When Adding a Book

1. Click **+ Add Book** â†’ **Upload Ebook** tab
2. Drop your file
3. BookShelf auto-fills title, author, cover from the file

### BookDrop (Automatic)

Drop files in a watched folder and BookShelf imports them automatically. See [Admin Guide](./admin-guide.md#bookdrop).
